# System Brief

## Nombre del sistema
Sistema de pedidos en tiendita de snacks

## Problema
Los pedidos se gestionan manualmente, lo que genera errores y desorden, por lo tanto esto da solución a ese problema

## Solución
Una API que permite registrar, consultar y actualizar pedidos de forma digital.

## Alcance (Scope)
- Listado de pedidos
- Crear pedidos
- Consultar pedidos
- Actualizar estado

## No Alcance (No Scope)
- Pagos
- Usuarios
- Base de datos persis
