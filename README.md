#  Sistema de Pedidos TIENDITA API

API REST desarrollada con FastAPI para gestionar pedidos de una tiendita.  
Permite crear pedidos, consultarlos, actualizar su estado y listarlos.

---

#  Descripción

Este sistema resuelve el problema de la gestión manual de pedidos en una tienda de snacks, permitiendo llevar un control digital básico mediante una API.

El MVP implementa el flujo principal del negocio:

- Crear pedidos
- Consultar pedidos
- Actualizar estado de pedidos
- Listar pedidos

---

#  Tecnologías utilizadas

- Python
- FastAPI
- Uvicorn
- Pydantic
